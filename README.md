# Traceback
### AI-powered Student Learning Diagnostics System

Traceback analyzes student behavioral signals to diagnose 
**why** students are struggling — not just that they are.

## Live Demo
- [Professor Dashboard](https://yourusername.github.io/Traceback/dashboard.html)
- [Student Portal](https://yourusername.github.io/Traceback/student.html)

## What it does
- Analyzes 28,785 students from the OULAD dataset
- Classifies students into 5 root causes of struggle
- Generates personalized 4-step learning plans
- 92.5% classifier accuracy (XGBoost)

## Root Causes
| Cause | Description |
|-------|-------------|
| NO_ENGAGEMENT | Attendance problem, not ability |
| DECLINING | Something changed recently |
| EXAM_ANXIETY | Knows material, struggles under pressure |
| NEEDS_SUPPORT | Close to passing, needs a push |
| KNOWLEDGE_GAP | Foundational concept missing |

## Tech Stack
Python · pandas · XGBoost · scikit-learn · HTML/CSS/JS
